# FalconPy 快捷键说明

## 菜单快捷键

### 文件操作
- **Ctrl+I** - 导入收藏夹
- **Ctrl+E** - 导出收藏夹  
- **Ctrl+Q** - 退出应用程序

### 视图操作
- **F11** - 切换全屏模式
- **F5** - 刷新当前内容

### 工具操作
- **Ctrl+L** - 账号登录对话框
- **Ctrl+,** - 打开设置对话框

## 图片操作快捷键

### 搜索功能
- **Ctrl+F** - 快速聚焦到搜索框并选中所有文本

### 图片管理
- **Ctrl+D** - 切换当前选中图片的收藏状态
- **Ctrl+S** - 下载当前选中的图片

### 页面导航
- **Ctrl+Left** - 转到上一页
- **Ctrl+Right** - 转到下一页

### 通用操作
- **Escape** - 退出全屏模式或关闭对话框

## 使用说明

### 图片选择
在使用图片相关快捷键（Ctrl+D, Ctrl+S）之前，需要先选择一张图片：
1. 点击任意图片缩略图来选择它
2. 选中的图片会显示蓝色边框
3. 然后可以使用快捷键对选中的图片进行操作

### 搜索功能
1. 使用 **Ctrl+F** 快速聚焦到搜索框
2. 输入搜索关键词
3. 按 **Enter** 或点击搜索按钮执行搜索

### 页面导航
- 在有搜索结果的情况下，使用 **Ctrl+Left/Right** 可以快速翻页
- 页面信息会显示在图片网格下方

### 全屏模式
- 使用 **F11** 进入/退出全屏模式
- 在全屏模式下，按 **Escape** 也可以退出全屏

## 注意事项

1. **图片操作前置条件**：收藏和下载功能需要先选择图片
2. **网络连接**：搜索和下载功能需要网络连接
3. **登录状态**：某些网站的高级功能可能需要登录
4. **快捷键冲突**：如果系统或其他软件占用了相同快捷键，可能会导致功能失效
